<h3>Meal Recommendation System REST API</h3>
<p>Here is a Meal Recommendation System REST API that operate on three Allergies</p>
<ul>
<li>Nut Allergy</li>
<li>ShellFish Allergy</li>
<li>SeaFood Allergy</li>
</ul>
<p>
Every user on the system is expected to pick their allergy from the Allergies operated on the system.
</p>
<p>
The system recommend meals according to the allergy picked by a user. The meals recommended include the Main Meal and two side meals. <br> The system can also fetch meal recommendations for more than one user at a time. 
</p>
